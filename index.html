CRESORY

Cresory is a gamification project that introduces children to financial education.
We have launched a five-level MVP with multiple videos, games and quizzes that explain concepts such as the history of money, saving, budgeting, investing, budget management, etc.

KAMU'S OFFERING

"Kamu's Offering" is a 2D platform game built around mechanics based on the interaction with a sphere, which is thrown and bounced to propel the main character across the stage.
It takes place in an African setting and tells a story about the love between two gods.
This is my first release on Steam (May 2021).

"Pop. 1280" is a narrative game based on the Spanish post-Civil War period.
